# DeFlock Design

The **$DeFlock** community's design system: a protest-print visual identity
(riso/screen-print ephemera) packaged as design tokens, React components, ready-made
templates, and machine-checkable brand rules — built so you can hand this repo to an AI
agent and get on-brand artwork back.

$DeFlock is a community-made Solana memecoin supporting the anti-surveillance movement
around [deflock.org](https://deflock.org), the open-source OpenStreetMap-based map of
Flock Safety ALPR cameras. **$DeFlock is not affiliated with, sponsored by, or endorsed
by deflock.org or the DeFlock open-source project.**

Voice in one line: *declarative watchdog.* "THEY MAP YOUR PLATES. WE MAP THE CAMERAS."

## Who this is for

- **Community members with an AI agent** (Claude, GPT, Gemini — anything). Clone the
  repo, point your agent at it ("read SKILL.md, then make me a sticker"), post the
  result. The rules travel with the repo; your agent enforces them.
- **Designers** — `guidelines/` holds 23 visual specimen cards; `readme.md` (this file)
  is the full design guide.
- **Developers** — `AGENTS.md` is the implementation handoff; `components/` +
  `styles.css` + the ui kit are the working parts.

## Make your first artifact (5 minutes)

1. **Serve the repo** from its root: `npx serve` (or `python3 -m http.server`). The
   cards, kit, and templates fetch local files and CDN scripts — opening them straight
   from disk (`file://`) shows a blank page.
2. **Look around:** open `/ui_kits/website/index.html` in the browser — that's the whole
   site rebuilt on this system. Add `?presscheck` to watch the brand audit run.
3. **Start from a template:** copy `templates/share-card/ShareCard.dc.html` (a 1080×1080
   social artifact with the brand check pre-wired) and edit the copy and art.
4. **Check it:** in the browser console run `pressCheck({mode: "expressive"})` — it
   prints a per-rule table and stamps **PASSED THE PRESS** or **PULLED**. Fix, re-run.
5. **Ship it** once it passes. Marked artifacts carry the exact line
   `COMMUNITY-MADE · UNAFFILIATED WITH DEFLOCK.ORG` — that's the deal (see
   [LICENSE-assets.md](LICENSE-assets.md)).

Working with an AI agent instead? Just tell it: *"Read SKILL.md and make me a
[sticker / share card / poster / page]."* SKILL.md contains the same flow, written for
agents.

## Toolchain notes

- Token pipeline (plain Node ≥18, zero npm deps, run from repo root):
  `node scripts/generate-tokens.mjs` (compile) · `node scripts/verify-tokens.mjs`
  (drift guard — also runs in CI).
- Consuming a project: link `styles.css`, self-host the fonts in `assets/fonts/`,
  compose the components in `components/`, gate artifacts with `register/press-check.js`.

## Hard doctrine (verbatim law — tokens.json.doctrine is canonical)
1. NO camera damage/destruction imagery or language, ever; cameras always intact.
2. Clean vs marked sets; marked ("COMMUNITY-MADE · UNAFFILIATED WITH DEFLOCK.ORG") is default with $DEFLOCK.
3. No URLs on-screen except inside the disclaimer wording.
4. No emdashes in artifact copy.
5. Operator-verified facts only — never invent amounts, deadlines, prices, vendors, rules.
6. The deflock.org blue crow is never community art; no single-bird marks — sole exception: the camera-headed crow mascot (ratified 2026-07-14).
7. Camera depictions follow the locked anatomy (`assets/cameras/*.svg`).

## CONTENT FUNDAMENTALS
- **Register**: declarative watchdog — second person, present tense, slightly ominous.
  Slam lines aim at the surveilled: "THEY MAP YOUR PLATES." / "WE MAP THE CAMERAS." /
  "THE FLOCK WATCHES BACK." First-person pledges run sentence case ("I support the right
  to privacy by holding $DeFlock.").
- **Case follows speech act** (register rule R5): uppercase = watchdog voice; sentence
  case = pledges + body. Uppercase is CSS `text-transform`, never typed.
- **Names**: `$DeFlock` in sentences; `$DEFLOCK` as slam/ticker — one line, never broken,
  hyphenated, or stacked; `dflock` never user-facing.
- **Separators**: `·` or periods. **No emdashes** in artifact copy. No emoji anywhere.
- **Sound words** (confirmations, stamps): only CLICK · PING · PINNED · FLAGGED ·
  SPOTTED · LOGGED. Puns are seasoning (stamps/stickers/chrome), never the entree.
- **Honesty grammar**: unverifiable numbers render as "—" with the reason ("live from
  Solana RPC"), estimates are labeled ("mark-to-market"), receipts carry provenance
  ("$DeFlock community confirmed…"). Errors are watchdog-flat: "QUERY DID NOT LAND".
  Button copy is a short command: CHECK MAP · WATCH · PIN IT · OPEN ON X.
- Every marked surface carries the exact disclaimer + "Not financial advice."

## VISUAL FOUNDATIONS
- **Colors**: three plates only — paper `#f2ede3` (~70% of any frame), ink `#14100a`
  (never pure black), red `#d92b1c` (THE accent: 1–2 hits, <15%, never on ink), plus
  gadsden `#e8b63a` (sanctioned lineage; in web UI: accent washes/chips, ink text only).
  No other colors, no gradients as decoration (only the shipped directional gadsden/red
  corner washes on panels), imagery is warm paper/ink duotone.
- **Type**: Archivo Black (slam, uppercase, leading .92–.98, −0.01em), Oswald (labels
  500/700 uppercase +.08em; body sentence-case), Space Mono (meta/chrome 400/700
  uppercase, +.12–.18em, tabular numerals). Italics never; `font-synthesis: none`.
  Page-title scale is ratified: `clamp(1.75rem, 4vw, 3rem)` + misregistration shadow.
- **Texture** (six locked recipes only): 9px halftone (alpha .10–.16), fractalNoise grain
  (.05–.08, multiply, fixed overlay), misregistration `.045em .03em 0 red` (display type
  only, red down-right), stamp (3px red border + inset paper/red rings, −4°..4°), torn
  edge (horizontal strips only), reg marks (1.5px crosshairs). UI mode stays at budget
  floors (R6).
- **Backgrounds**: halftone-dotted paper everywhere; panels are warm white-mixes
  (92/94/86%) with 1.5px ink hairlines; grounding strips use ink-8% or gadsden-14/18% washes.
- **Borders**: 1.5px ink hairline is law (3px heavy for feature panels; dashed = empty
  states). Corner radius 0 — the avatar circle is the sole exception.
- **Shadows**: hard offsets only, never blurred: 2.8px (accent buttons), 6px (raised
  panels), hover 2.4px at 36% ink. No soft elevation system.
- **Interaction**: hover = translate(−1px,−1px) + hard shadow; press = scale(.96) (icons
  .92), 120ms ease-out, transform-only; focus = 2px ink outline offset 2; disabled =
  opacity .55. Reduced motion collapses everything to CUT (brand-true: riso has no tweens).
- **Motion**: PRESS/REWARD/PLATE/CUT eases at 24fps; REWARD is the only overshoot; grain
  steps every 2–3 frames. Web UI micro-interactions are 120ms; no infinite loops.
- **Layout**: 8px base unit; chrome margin 4.5% of short side; content column 1184px;
  sticky header (mobile: fixed bottom tab bar ≤720px); square-native-first for artifacts.
- **Tilt discipline**: expressive artifacts lean (≥2 rotated, opposing); UI containers sit
  straight — tilt only on stamp-class elements (badges, active nav, one accent button).

## ICONOGRAPHY
- **UI icons**: [Lucide](https://lucide.dev) — production uses `lucide-react`
  (ExternalLink, ArrowUpRight, CheckCircle2, FileText, ShieldCheck); kits load the lucide
  UMD from CDN and render `<i data-lucide="…">`. 14–16px inline, stroke inherits ink.
- **Brand glyphs**: copied SVGs in `assets/` — breakthrough emblem (community mark/pfp),
  coin badge (marked set only, NEVER a pfp), murmuration motifs (texture), spider-eye
  camera face (villain glyph — subject, never identity), camera pod profile + spider-eye front (subject; wide-silhouette retired 2026-07-14, operator preference).
  **Copy, never redraw** — geometry source of truth.
- **Mascot**: camera-headed crow, remade 2026-07-14 into the plates (`assets/mascot/`):
  halftone `deflock-mascot-plates-1024.png` (+duotone), `deflock-mascot-pfp-400.png`,
  flat icon, re-inked lineart badge. The superseded blue-era renders are kept upstream
  for provenance and are not part of this repo.
- No icon font, no emoji, no unicode-as-icon (arrows ‹ › allowed on icon buttons).
