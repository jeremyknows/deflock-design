The single $DeFlock button — use it for every action; never hand-roll button CSS (that is how the site got 6 divergent copies).

```jsx
import { Button } from "./Button";
<Button variant="primary" onClick={go}>Check map</Button>
<Button href={X_URL} target="_blank" variant="accent">Community X</Button>
<Button size="xl" type="submit">CHECK MAP</Button>   // hero form: Oswald label type
<Button iconOnly aria-label="Previous">‹</Button>
```

Variants: `primary` (ink), `secondary` (paper, default), `accent` (gadsden stamp: hard ink shadow + −1.5° tilt — max ONE per view, register R6), `quiet` (ink-8% wash, for tertiary link-chips). Sizes sm/md/lg/xl = 40/44/48/56 min-height; 44 is the tap-target floor. Disabled = opacity .55. Copy on buttons is uppercase, watchdog-short ("CHECK MAP", "PIN IT", never "Click here to…").
