Stamped chip/badge — kickers, role badges, showcase stamps, filter chips. The ONLY tilted UI elements (register R6).

```jsx
import { Badge } from "./Badge";
<Badge variant="kicker">WHO WATCHES THE WATCHERS</Badge>
<Badge variant="stamp">HOLDER</Badge>
<Badge variant="solid">Verified claimed</Badge>
<Badge variant="accent" >4,096 $DEFLOCK</Badge>
<Badge variant="chip" selected>STICKERS</Badge>
```

kicker = red ring + ink text (section eyebrows; red type never at meta size — 4.2:1 fails AA); stamp = double ink ring (roles/status); solid = ink fill (showcase); accent = gadsden value chip (straight); chip = filter (selected → ink fill). Copy is short uppercase meta. Never stack two tilted badges side by side with the same sign — opposing or straight.
